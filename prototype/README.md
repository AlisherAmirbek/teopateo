# TeoPateo UI Prototype

This is a temporary browser prototype for the iOS app UI while Xcode is not ready.

Open `index.html` directly in a browser, or serve this folder locally:

```bash
python3 -m http.server 5173
```

Then visit:

```text
http://localhost:5173/prototype/
```

The prototype covers:

- Today dashboard
- Craving emergency mode
- Quit plan
- Daily check-in
- Slip recovery
- Pattern insights
- Basic AI coach mock
