# Calculated Insights

## Goal

Replace static dashboard and insight values with calculations from stored check-ins and craving events.

## Calculations

- Smoke-free streak.
- Cravings handled.
- Cigarettes avoided.
- Money saved.
- Highest-risk time windows.
- Most frequent triggers.
- Suggested plan adjustments.

## App Impact

The Today and Insights tabs should reflect the user's real history instead of fixed sample values.
