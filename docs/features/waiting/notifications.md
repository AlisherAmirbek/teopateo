# Notifications

## Goal

Add local notifications for risky windows and daily check-ins.

## Notification Types

- Morning plan reminder.
- Risky-window warning.
- Post-meal reminder.
- Evening check-in.
- Medication or nicotine replacement reminder if the user enables it later.

## App Impact

Notifications should be opt-in, specific, and based on the user's quit plan and trigger history.
