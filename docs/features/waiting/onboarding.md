# Onboarding

## Goal

Capture the user's baseline smoking context and turn it into the first version of their quit plan.

## Inputs

- Cigarettes per day.
- Common smoking times.
- Main triggers.
- Quit date.
- Reasons for quitting.
- Support contact.

## App Impact

The onboarding flow should populate the quit plan, initial dashboard metrics, craving-mode motivation, and support circle.
