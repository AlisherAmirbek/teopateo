# Event Saving

## Goal

Make check-in submission and craving completion create durable records.

## Events

- Daily check-in saved from mood, stress, confidence, smoke/no-smoke status, focus note, and slip note.
- Craving event saved from timer completion, selected triggers, and completion outcome.

## App Impact

The app should treat each saved event as source data for dashboard metrics, insights, slip recovery, and coach context.
